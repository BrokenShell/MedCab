# MedCab
## Cannabis Suggestion Bot
### Robert Sharp
- August 2020


## User Input: String
- Random Strain
- Strain id
- Strain Name
- Desired Effect
- Desired Flavor
- Desired Type

## Strain Object: JSON Dictionary
- Strain Name
- Strain Type
- User Rating (0.0 - 5.0)
- Effects List
- Flavors List
- Description Text
- Index Number (0 - 2154)
